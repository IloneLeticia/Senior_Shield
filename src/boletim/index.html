Gerador de Boletim

Como usar — siga os passos abaixo:

  1. Passo 1: Preencha todos os campos do formulário com suas informações.
  2. Passo 2: Clique no botão azul "Gerar Boletim".
  3. Passo 3: O texto do boletim aparecerá à direita. Você pode copiar ou salvar em PDF.
  4. Passo 4: Clique em "Ir para a Delegacia Virtual" e use o texto gerado para registrar a ocorrência.
